INFORMATION ABOUT HTML

What is HTML?

HTML, or HyperText Markup Language, is the standard markup language used to create and structure web pages. It serves as the foundation for nearly all websites, defining the content and organization of web pages.

Mission:

Key aspects of HTML's mission include:


Structuring Web Content


Enabling Hypertext Navigation


Facilitating Content Accessibility


Providing a Foundation for Web Development

history:

The Timline of HTML 1991- Tim Berners-Lee invents HTML 1.0

1993- HTML 1.0 is released. Not many developers are creating websites at this time.

1995- HTML 2.0 is published. This remained the standard markup language for designing and creating websites until 1997.

1997- HTML 3.0 was invented. Here, Dave Raggett introduced a fresh draft on HTML, which improved new features of HTML and gave more powerful characteristics for webmasters in designing websites. Unfortunately, the powerful features slowed down the browser in applying further improvements.

1999- The widely-used HTML 4.0 comes out. It is very successful.

2014- HTML 5.0 is released and used worldwide. It is said to be the extended version of HTML 4.01 which was published in 2012.

vision:

Universal Accessibility: HTML was designed to be platform-neutral, ensuring that web content could be viewed and accessed by anyone, regardless of their operating system or device.

Structured Content: It provides a basic set of tags and elements to define the structure and semantics of a document, such as headings, paragraphs, lists, and links, making content organized and machine-readable.

Interconnectivity: The core concept of hypertext enables linking between documents, forming a vast, interconnected network of information that facilitates easy navigation and discovery.

Simplicity and Extensibility: HTML was intended to be relatively easy to learn and use, while also being extensible enough to accommodate future needs and evolving web technologies.
